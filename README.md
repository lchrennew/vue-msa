# Vue-MSA

Vue Micro Service Architecture Framework.


## HOWTO Start

### Prerequirement
- node.js (Latest version is strongly recommended)
- yarn (`npm i -g yarn`)
- web browser (Chrome is recommended)
- IDE is optional

### Install dependencies and start the demo
``` shell
cd sub-app2
yarn
yarn dev


cd ../root-app
yarn
yarn dev

```
